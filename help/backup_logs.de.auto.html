Sicherungsprotokolle
Bei der Suche können Sie die Begriffe „Feld:Wert“ verwenden, um Sicherungsprotokolle präzise zu filtern, indem Sie mehrere Feldbegriffe durch Kommas kombinieren.

Unterstützte Felder
Feld Beschreibung Beispiel
user Übereinstimmung mit Benutzer user:^username$
domain Abgleich mit Domänenliste domain:^parent.tld
dest Zielpfad abgleichen dest:/path/to/file
desc Beschreibung des Spielplans desc:wild echo
time entspricht formatiertem Datum time:06:00 PM
type incremental oder full type:incremental
size Vergleiche mit Einheiten size:>40m<55m
status Statusbeschriftungstext prüfen status:failed
