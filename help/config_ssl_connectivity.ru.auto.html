Проверьте перед запросом сертификата
Этот параметр определяет, какие проверки подключения выполняются по умолчанию при запросе SSL-сертификата у провайдера, чтобы избежать неудачных запросов для доменов, которые не настроены должным образом. Доступны следующие варианты:

  • Проверка подключения . Проверьте, может ли домен принимать HTTP-подключения из остального Интернета, используя службу проверки подключения Virtualmin.
  • Проверка домена . Запуск проверок конфигурации Virtualmin для веб-сайта и записей DNS.
  • Пропустить тесты - Тестирование не проводится.
