URL adresa služby HTTP na detekciu externej IP adresy systému
Toto pole použite na vyhľadanie verejných IP adries vášho servera IPv4 a IPv6 pomocou webovej služby. Ak váš server používa IPv6, zadajte dve URL adresy oddelené medzerou: jednu iba pre IPv4 a jednu iba pre IPv6. Ak nie, zadajte iba URL adresu IPv4.

Predvolený pár je http://v4.software.virtualmin.com/cgi-bin/ip.cgi http://v6.software.virtualmin.com/cgi-bin/ip.cgi . Môžete tiež použiť iné služby, ako napríklad https://api.ipify.org https://api6.ipify.org .
