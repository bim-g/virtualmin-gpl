Activer MTA-STS pour ce domaine ?
MTA-STS est un protocole permettant d'informer les autres serveurs de messagerie que les e-mails envoyés via SMTP à ce domaine doivent être acheminés uniquement via un transport sécurisé. Lorsque cette option est activée, Virtualmin crée les enregistrements DNS, la configuration du serveur web et le certificat SSL (si possible) nécessaires pour permettre la distribution chiffrée des e-mails entrants.
