Verificar antes de solicitar el certificado
Esta opción determina qué comprobaciones de conectividad se realizan por defecto al solicitar un certificado SSL a un proveedor, para evitar solicitudes fallidas en dominios mal configurados. Las opciones son:

  • Probar la conectividad : pruebe que el dominio pueda aceptar conexiones HTTP desde el resto de Internet, utilizando el servicio de verificación de conectividad de Virtualmin.
  • Validar dominio : ejecuta las comprobaciones de validación de configuración de Virtualmin para el sitio web y los registros DNS.
  • Omitir pruebas : no se realiza ninguna prueba.
