Ermitteln Sie die externe IP-Adresse des Systems mithilfe von DNS
Diese Option verwendet den Befehl dig , um einen DNS-Server abzufragen und die externe IP-Adresse des Systems zu ermitteln. Sie können dieses Feld beispielsweise auf myip.opendns.com resolver1.opendns.com setzen (durch Leerzeichen getrennter Abfragename und Nameserver). Dadurch wird die Verwendung einer HTTP-basierten Methode vermieden und die IP-Adresse direkt vom DNS-Server abgerufen.

Die DNS-Methode sollte jedoch mit Vorsicht verwendet werden, wenn sich das System hinter einem NAT, Proxy oder VPN befindet, da die erkannte IP möglicherweise nicht der tatsächlichen öffentlichen IP des Computers entspricht. Stattdessen wird möglicherweise die externe IP des NAT-Gateways, Proxys oder VPN-Endpunkts angezeigt.
