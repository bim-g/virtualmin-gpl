Nom d'utilisateur de base de l'administrateur
La partie de base du nom d'utilisateur que l'administrateur supplémentaire utilisera pour se connecter, à laquelle le nom de domaine sera ajouté. Ce nom de compte est soumis aux mêmes restrictions que tout autre compte du système (c.-à-d. pas d'espaces ni de caractères spéciaux autres que ceux autorisés dans les noms d'utilisateur).
