Adres URL usługi HTTP do wykrywania zewnętrznego adresu IP systemu
Użyj tego pola, aby znaleźć publiczne adresy IP serwera (IPv4 i IPv6) za pomocą usługi sieciowej. Jeśli serwer korzysta z protokołu IPv6, wprowadź dwa adresy URL oddzielone spacją: jeden dla protokołu IPv4 i jeden dla protokołu IPv6. W przeciwnym razie wprowadź tylko adres URL protokołu IPv4.

Domyślna para to http://v4.software.virtualmin.com/cgi-bin/ip.cgi http://v6.software.virtualmin.com/cgi-bin/ip.cgi . Możesz również skorzystać z innych usług, takich jak https://api.ipify.org https://api6.ipify.org .
