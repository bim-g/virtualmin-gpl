Zálohovacie protokoly
Pri vyhľadávaní môžete použiť výrazy „field:value“ na presné filtrovanie protokolov zálohovania, pričom viacero výrazov z polí môžete oddeliť čiarkami.

Podporované polia
Pole Popis Príklad
user zhoda s používateľom user:^username$
domain zhoda so zoznamom domén domain:^parent.tld
dest zhoda s cieľovou cestou dest:/path/to/file
desc popis rozpisu zápasov desc:wild echo
time zhoduje sa s formátovaným dátumom time:06:00 PM
type incremental alebo full type:incremental
size porovnania s jednotkami size:>40m<55m
status text označenia stavu kontroly status:failed
