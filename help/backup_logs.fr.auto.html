Journaux de sauvegarde
Lors de la recherche, vous pouvez utiliser les termes « champ : valeur » pour filtrer précisément les journaux de sauvegarde, en combinant plusieurs termes de champ avec des virgules.

Champs pris en charge
Champ Description Exemple
user correspondance sur l'utilisateur user:^username$
domain correspondance avec la liste de domaines domain:^parent.tld
dest chemin de destination de correspondance dest:/path/to/file
desc description du calendrier des matchs desc:wild echo
time correspond à la date formatée time:06:00 PM
type incremental ou full type:incremental
size comparaisons avec des unités size:>40m<55m
status vérifier le texte de l'étiquette d'état status:failed
