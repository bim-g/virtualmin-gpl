Inclure le nom de domaine dans les noms d'utilisateur d'administrateur supplémentaires ?
Cette option sélectionne le mode de génération des noms d'utilisateur pour les administrateurs supplémentaires du domaine. Par défaut, Virtualmin utilise le nom d'utilisateur de base demandé et signale une erreur en cas de conflit avec un autre domaine. Cependant, si vous sélectionnez « Oui », la première partie du nom d'utilisateur sera ajoutée au début ou à la fin du domaine, comme défini dans l'option de configuration « Style du nom de domaine dans les noms d'utilisateur ».
