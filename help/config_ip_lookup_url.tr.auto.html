Sistemin harici IP tespiti için HTTP servis URL'si
Sunucunuzun genel IP'lerini (IPv4 ve IPv6) bir web hizmeti kullanarak bulmak için bu alanı kullanın. Sunucunuz IPv6 kullanıyorsa, aralarında boşluk bırakarak iki URL girin: biri yalnızca IPv4, diğeri yalnızca IPv6 için. Kullanmıyorsa, yalnızca IPv4 URL'sini girin.

Varsayılan çift http://v4.software.virtualmin.com/cgi-bin/ip.cgi http://v6.software.virtualmin.com/cgi-bin/ip.cgi şeklindedir. https://api.ipify.org https://api6.ipify.org gibi diğer servisleri de kullanabilirsiniz.
