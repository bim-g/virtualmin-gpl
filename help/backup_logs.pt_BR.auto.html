Logs de backup
Ao pesquisar, você pode usar termos "campo:valor" para filtrar logs de backup com precisão, combinando vários termos de campo com vírgulas.

Campos suportados
Campo Descrição Exemplo
user correspondência no usuário user:^username$
domain correspondência com lista de domínios domain:^parent.tld
dest caminho de destino correspondente dest:/path/to/file
desc descrição do cronograma de partidas desc:wild echo
time corresponde à data formatada time:06:00 PM
type incremental ou full type:incremental
size comparações com unidades size:>40m<55m
status verificar texto do rótulo de status status:failed
