Controllare prima di richiedere il certificato
Questa opzione determina quali controlli di connettività vengono eseguiti per impostazione predefinita quando si richiede un certificato SSL a un provider, per evitare richieste non riuscite per domini non configurati correttamente. Le opzioni sono:

  • Test di connettività : verifica che il dominio possa accettare connessioni HTTP dal resto di Internet, utilizzando il servizio di controllo della connettività di Virtualmin.
  • Convalida dominio : esegui i controlli di convalida della configurazione di Virtualmin per il sito web e i record DNS.
  • Salta i test : non viene eseguito alcun test.
