Sikkerhetskopieringslogger
Når du søker, kan du bruke termene «field:value» for å filtrere sikkerhetskopieringslogger presist, ved å kombinere flere felttermer med komma.

Støttede felt
Felt Beskrivelse Eksempel
user samsvar på bruker user:^username$
domain samsvar mot domenelisten domain:^parent.tld
dest samsvar med destinasjonsstien dest:/path/to/file
desc beskrivelse av kampprogrammet desc:wild echo
time samsvarer med formatert dato time:06:00 PM
type incremental eller full type:incremental
size sammenligninger med enheter size:>40m<55m
status sjekk statusetiketttekst status:failed
