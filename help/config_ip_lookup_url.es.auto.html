URL del servicio HTTP para la detección de IP externa del sistema
Utilice este campo para buscar las direcciones IP públicas IPv4 e IPv6 de su servidor mediante un servicio web. Si su servidor usa IPv6, introduzca dos URL separadas por un espacio: una solo para IPv4 y otra solo para IPv6. De lo contrario, introduzca solo la URL IPv4.

El par predeterminado es http://v4.software.virtualmin.com/cgi-bin/ip.cgi http://v6.software.virtualmin.com/cgi-bin/ip.cgi . También puede usar otros servicios como https://api.ipify.org https://api6.ipify.org
