システム外部 IP 検出用の HTTP サービス URL
このフィールドを使用して、ウェブサービスを利用してサーバーのパブリックIPアドレス（IPv4およびIPv6）を検索できます。サーバーがIPv6を使用している場合は、スペースで区切って2つのURLを入力してください。1つはIPv4専用、もう1つはIPv6専用です。そうでない場合は、IPv4 URLのみを入力してください。

デフォルトのペアはhttp://v4.software.virtualmin.com/cgi-bin/ip.cgi http://v6.software.virtualmin.com/cgi-bin/ip.cgiです。 https://api.ipify.org https://api6.ipify.orgなどの他のサービスも使用できます。
