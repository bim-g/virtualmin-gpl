HTTP-Dienst-URL zur Erkennung externer IP-Adressen des Systems
Verwenden Sie dieses Feld, um die öffentlichen IP-Adressen Ihres Servers (IPv4 und IPv6) über einen Webdienst zu ermitteln. Wenn Ihr Server IPv6 verwendet, geben Sie zwei durch ein Leerzeichen getrennte URLs ein: eine für reines IPv4 und eine für reines IPv6. Andernfalls geben Sie nur die IPv4-URL ein.

Das Standardpaar ist http://v4.software.virtualmin.com/cgi-bin/ip.cgi http://v6.software.virtualmin.com/cgi-bin/ip.cgi . Sie können auch andere Dienste wie https://api.ipify.org https://api6.ipify.org verwenden.
