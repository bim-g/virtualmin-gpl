URL du service HTTP pour la détection d'adresse IP externe du système
Utilisez ce champ pour trouver les adresses IP publiques IPv4 et IPv6 de votre serveur via un service web. Si votre serveur utilise IPv6, saisissez deux URL séparées par un espace : une pour IPv4 uniquement et une pour IPv6 uniquement. Sinon, saisissez uniquement l'URL IPv4.

La paire par défaut est http://v4.software.virtualmin.com/cgi-bin/ip.cgi http://v6.software.virtualmin.com/cgi-bin/ip.cgi . Vous pouvez également utiliser d'autres services comme https://api.ipify.org https://api6.ipify.org .
