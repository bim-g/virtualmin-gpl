Имя пользователя базы администратора
Базовая часть имени пользователя, которую дополнительный администратор будет использовать для входа в систему, к которой будет добавлено доменное имя. На это имя учётной записи распространяются те же ограничения, что и на любую другую учётную запись в системе (то есть, без пробелов и специальных символов, кроме разрешенных в именах пользователей).
