Habilitar MTA-STS para este domínio?
O MTA-STS é um protocolo que informa outros servidores de e-mail que e-mails enviados via SMTP para este domínio devem ser entregues somente por meio de um transporte seguro. Quando esta opção estiver habilitada, o Virtualmin criará os registros DNS, a configuração do servidor web e o certificado SSL necessários (se possível) para permitir a entrega criptografada de e-mails recebidos.
