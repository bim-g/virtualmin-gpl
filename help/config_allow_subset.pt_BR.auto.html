Excluir nomes de domínio não verificáveis?
Por padrão, quando esta opção está definida como "Não", uma solicitação de certificado SSL de um provedor ACME como o Let's Encrypt só será bem-sucedida se todos os nomes de domínio puderem ser validados. Alterá-la para "Sim" permite que a solicitação seja parcialmente bem-sucedida, excluindo do certificado nomes de domínio que não podem ser verificados.
