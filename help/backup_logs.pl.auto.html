Dzienniki kopii zapasowych
Podczas wyszukiwania możesz użyć wyrażeń „pole:wartość”, aby precyzyjnie filtrować dzienniki kopii zapasowych, łącząc wiele wyrażeń pól przecinkami.

Obsługiwane pola
Pole Opis Przykład
user dopasuj do użytkownika user:^username$
domain dopasuj do listy domen domain:^parent.tld
dest dopasuj ścieżkę docelową dest:/path/to/file
desc opis harmonogramu meczu desc:wild echo
time pasuje do sformatowanej daty time:06:00 PM
type incremental lub full type:incremental
size porównania z jednostkami size:>40m<55m
status sprawdź tekst etykiety statusu status:failed
