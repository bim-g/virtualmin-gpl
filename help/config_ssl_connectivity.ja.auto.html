証明書の申請前に確認する
このオプションは、プロバイダーからSSL証明書を要求する際に、適切に設定されていないドメインへの要求が失敗することを回避するために、デフォルトで実行される接続チェックの種類を決定します。オプションは以下のとおりです。

  • 接続性をテストする- Virtualmin の接続チェック サービスを使用して、ドメインがインターネットの他の部分からの HTTP 接続を受け入れることができるかどうかをテストします。
  • ドメインの検証- ウェブサイトと DNS レコードに対して Virtualmin の構成検証チェックを実行します。
  • テストをスキップ- テストは実行されません。
