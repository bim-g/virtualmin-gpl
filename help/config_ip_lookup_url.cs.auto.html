URL adresa HTTP služby pro detekci externí IP adresy systému
Toto pole použijte k nalezení veřejných IP adres vašeho serveru (IPv4 a IPv6) pomocí webové služby. Pokud váš server používá IPv6, zadejte dvě adresy URL oddělené mezerou: jednu pouze pro IPv4 a jednu pouze pro IPv6. Pokud ne, zadejte pouze adresu URL pro IPv4.

Výchozí pár je http://v4.software.virtualmin.com/cgi-bin/ip.cgi http://v6.software.virtualmin.com/cgi-bin/ip.cgi . Můžete také použít další služby, jako například https://api.ipify.org https://api6.ipify.org .
