Zistenie externej IP adresy systému pomocou DNS
Táto možnosť používa príkaz dig na dotazovanie DNS servera a zistenie externej IP adresy systému. Napríklad, toto pole môžete nastaviť na myip.opendns.com resolver1.opendns.com (názov dotazu a nameserver oddelené medzerami). Tým sa zabráni spoliehaniu sa na metódu založenú na HTTP a IP adresa sa načíta priamo zo DNS servera.

Metóda DNS by sa však mala používať opatrne, ak je systém za NAT, proxy alebo VPN, pretože zistená IP adresa nemusí zodpovedať skutočnej verejnej IP adrese zariadenia. Namiesto toho môže zobrazovať externú IP adresu brány NAT, proxy alebo koncového bodu VPN.
