Резервные журналы
При поиске можно использовать термины «поле:значение» для точной фильтрации журналов резервного копирования, объединяя несколько терминов полей с помощью запятых.

Поддерживаемые поля
Поле Описание Пример
user совпадение по пользователю user:^username$
domain сопоставить со списком доменов domain:^parent.tld
dest сопоставить путь назначения dest:/path/to/file
desc описание расписания матчей desc:wild echo
time соответствует отформатированной дате time:06:00 PM
type incremental или full type:incremental
size сравнения с единицами size:>40m<55m
status текст метки статуса проверки status:failed
