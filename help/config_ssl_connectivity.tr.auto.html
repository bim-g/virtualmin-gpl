Sertifika talebinden önce kontrol edin
Bu seçenek, düzgün yapılandırılmamış alan adları için başarısız istekleri önlemek amacıyla bir sağlayıcıdan SSL sertifikası istendiğinde varsayılan olarak hangi bağlantı kontrollerinin gerçekleştirileceğini belirler. Seçenekler şunlardır:

  • Bağlantıyı test edin - Virtualmin'in bağlantı kontrol hizmetini kullanarak etki alanının internetin geri kalanından HTTP bağlantılarını kabul edip edemediğini test edin.
  • Etki alanını doğrula - Web sitesi ve DNS kayıtları için Virtualmin'in yapılandırma doğrulama kontrollerini çalıştır.
  • Testleri atla - Hiçbir test yapılmaz.
