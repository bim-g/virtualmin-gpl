Nicht überprüfbare Domänennamen ausschließen?
Wenn diese Option standardmäßig auf „Nein“ gesetzt ist, ist eine SSL-Zertifikatsanforderung von einem ACME-Anbieter wie Let’s Encrypt nur dann erfolgreich, wenn alle Domänennamen validiert werden können. Wenn Sie die Option auf „Ja“ setzen, ist die Anforderung teilweise erfolgreich, indem Domänennamen, die nicht verifiziert werden können, vom Zertifikat ausgeschlossen werden.
