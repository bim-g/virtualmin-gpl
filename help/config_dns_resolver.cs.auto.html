Zjištění externí IP adresy systému pomocí DNS
Tato možnost používá příkaz dig k dotazování DNS serveru a detekci externí IP adresy systému. Toto pole můžete například nastavit na myip.opendns.com resolver1.opendns.com (název dotazu a nameserver oddělený mezerami). Tím se vyhnete spoléhání se na metodu založenou na HTTP a IP adresa se načte přímo ze serveru DNS.

Metoda DNS by však měla být používána s opatrností, pokud je systém za NAT, proxy nebo VPN, protože zjištěná IP adresa nemusí odrážet skutečnou veřejnou IP adresu počítače. Místo toho může zobrazovat externí IP adresu brány NAT, proxy nebo koncového bodu VPN.
