Включить MTA-STS для этого домена?
MTA-STS — это протокол, информирующий другие почтовые серверы о том, что электронная почта, отправляемая по SMTP на этот домен, должна доставляться только по защищённому протоколу. При включении этой опции Virtualmin создаст необходимые записи DNS, конфигурацию веб-сервера и SSL-сертификат (если возможно) для обеспечения зашифрованной доставки входящей почты.
