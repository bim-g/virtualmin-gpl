URL do serviço HTTP para detecção de IP externo do sistema
Use este campo para encontrar os IPs públicos IPv4 e IPv6 do seu servidor usando um serviço web. Se o seu servidor usa IPv6, insira duas URLs separadas por um espaço: uma para IPv4 e outra para IPv6. Caso contrário, insira apenas a URL IPv4.

O par padrão é http://v4.software.virtualmin.com/cgi-bin/ip.cgi http://v6.software.virtualmin.com/cgi-bin/ip.cgi . Você também pode usar outros serviços, como https://api.ipify.org https://api6.ipify.org .
