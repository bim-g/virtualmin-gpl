系統外部IP偵測的HTTP服務URL
使用此欄位透過 Web 服務尋找伺服器的公用 IP（IPv4 和 IPv6）。如果您的伺服器使用 IPv6，請輸入兩個 URL（以空格分隔）：一個用於僅 IPv4 的 URL，另一個用於僅 IPv6 的 URL。如果不是，請僅輸入 IPv4 URL。

預設對是http://v4.software.virtualmin.com/cgi-bin/ip.cgi http://v6.software.virtualmin.com/cgi-bin/ip.cgi 。您也可以使用其他服務，例如https://api.ipify.org https://api6.ipify.org 。
