Wykryj zewnętrzny adres IP systemu za pomocą DNS
Ta opcja używa polecenia dig do zapytania serwera DNS i wykrycia zewnętrznego adresu IP systemu. Na przykład, można ustawić to pole na myip.opendns.com resolver1.opendns.com (nazwa zapytania i serwer nazw rozdzielone spacjami). Pozwala to uniknąć korzystania z metody opartej na protokole HTTP i pobrać adres IP bezpośrednio z serwera DNS.

Należy jednak zachować ostrożność podczas korzystania z metody DNS, jeśli system znajduje się za NAT, serwerem proxy lub siecią VPN, ponieważ wykryty adres IP może nie odzwierciedlać faktycznego publicznego adresu IP komputera. Zamiast tego może wskazywać zewnętrzny adres IP bramy NAT, serwera proxy lub punktu końcowego VPN.
