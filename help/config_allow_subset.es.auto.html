¿Excluir nombres de dominio no verificables?
De forma predeterminada, cuando esta opción está configurada en "No", una solicitud de certificado SSL de un proveedor ACME como Let's Encrypt solo se aceptará si todos los nombres de dominio se pueden validar. Cambiarla a "Sí" permite que la solicitud se acepte parcialmente, excluyendo del certificado los nombres de dominio que no se pueden verificar.
