Registri di backup
Durante la ricerca, è possibile utilizzare i termini "campo:valore" per filtrare con precisione i log di backup, combinando più termini di campo con virgole.

Campi supportati
Campo Descrizione Esempio
user corrispondenza sull'utente user:^username$
domain corrispondenza con l'elenco dei domini domain:^parent.tld
dest percorso di destinazione della corrispondenza dest:/path/to/file
desc descrizione del programma delle partite desc:wild echo
time corrisponde alla data formattata time:06:00 PM
type incremental o full type:incremental
size confronti con unità size:>40m<55m
status controlla il testo dell'etichetta di stato status:failed
