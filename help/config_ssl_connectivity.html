Check before certificate request
This option determines what connectivity checks are performed by default when requesting an SSL certificate from a provider, to avoid failed requests for domains that are not properly configured. The options are :

  • Test connectivity - Test that the domain can accept HTTP connections from the rest of the internet, using Virtualmin's connectivity check service.
  • Validate domain - Run Virtualmin's config validation checks for the website and DNS records.
  • Skip tests - No testing is done.
