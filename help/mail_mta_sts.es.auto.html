¿Habilitar MTA-STS para este dominio?
MTA-STS es un protocolo que informa a otros servidores de correo que el correo electrónico enviado vía SMTP a este dominio solo debe entregarse mediante un transporte seguro. Al habilitar esta opción, Virtualmin creará los registros DNS, la configuración del servidor web y el certificado SSL necesarios (si es posible) para habilitar la entrega cifrada del correo electrónico entrante.
