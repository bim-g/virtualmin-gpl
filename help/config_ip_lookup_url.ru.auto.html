URL-адрес службы HTTP для определения внешнего IP-адреса системы
Используйте это поле, чтобы найти публичные IP-адреса вашего сервера (IPv4 и IPv6) с помощью веб-сервиса. Если ваш сервер использует IPv6, введите два URL-адреса, разделённых пробелом: один для IPv4 и один для IPv6. В противном случае введите только URL-адрес IPv4.

Пара по умолчанию: http://v4.software.virtualmin.com/cgi-bin/ip.cgi http://v6.software.virtualmin.com/cgi-bin/ip.cgi . Вы также можете использовать другие сервисы, например https://api.ipify.org https://api6.ipify.org .
