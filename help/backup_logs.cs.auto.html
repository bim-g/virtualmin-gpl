Zálohovací protokoly
Při vyhledávání můžete k přesnému filtrování protokolů záloh použít výrazy „field:value“ a kombinovat více výrazů z polí čárkami.

Podporovaná pole
Pole Popis Příklad
user shoda s uživatelem user:^username$
domain porovnání se seznamem domén domain:^parent.tld
dest shoda s cílovou cestou dest:/path/to/file
desc popis rozpisu zápasů desc:wild echo
time odpovídá formátovanému datu time:06:00 PM
type incremental nebo full type:incremental
size srovnání s jednotkami size:>40m<55m
status text popisku stavu kontroly status:failed
