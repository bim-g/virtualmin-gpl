URL del servizio HTTP per il rilevamento dell'IP esterno del sistema
Utilizza questo campo per trovare gli IP pubblici IPv4 e IPv6 del tuo server tramite un servizio web. Se il tuo server utilizza IPv6, inserisci due URL separati da uno spazio: uno per IPv4 e uno per IPv6. In caso contrario, inserisci solo l'URL IPv4.

La coppia predefinita è http://v4.software.virtualmin.com/cgi-bin/ip.cgi http://v6.software.virtualmin.com/cgi-bin/ip.cgi . È possibile utilizzare anche altri servizi come https://api.ipify.org https://api6.ipify.org .
