Vérifier avant la demande de certificat
Cette option détermine les vérifications de connectivité effectuées par défaut lors de la demande d'un certificat SSL auprès d'un fournisseur, afin d'éviter l'échec des demandes pour les domaines mal configurés. Les options sont :

  • Tester la connectivité - Testez que le domaine peut accepter les connexions HTTP du reste d'Internet, à l'aide du service de vérification de connectivité de Virtualmin.
  • Valider le domaine - Exécutez les contrôles de validation de configuration de Virtualmin pour le site Web et les enregistrements DNS.
  • Sauter les tests - Aucun test n'est effectué.
