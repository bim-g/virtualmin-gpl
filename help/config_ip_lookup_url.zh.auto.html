系统外部IP检测的HTTP服务URL
使用此字段通过 Web 服务查找服务器的公网 IP（IPv4 和 IPv6）。如果您的服务器使用 IPv6，请输入两个 URL（以空格分隔）：一个用于仅 IPv4 的 URL，另一个用于仅 IPv6 的 URL。如果不是，请仅输入 IPv4 URL。

默认对是http://v4.software.virtualmin.com/cgi-bin/ip.cgi http://v6.software.virtualmin.com/cgi-bin/ip.cgi 。您还可以使用其他服务，例如https://api.ipify.org https://api6.ipify.org 。
