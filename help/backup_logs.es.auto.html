Registros de respaldo
Al realizar una búsqueda, puede utilizar los términos "campo:valor" para filtrar los registros de respaldo con precisión, combinando varios términos de campo con comas.

Campos admitidos
Campo Descripción Ejemplo
user coincidencia en el usuario user:^username$
domain coincidencia con la lista de dominios domain:^parent.tld
dest ruta de destino del partido dest:/path/to/file
desc Descripción del calendario de partidos desc:wild echo
time coincide con la fecha formateada time:06:00 PM
type incremental o full type:incremental
size comparaciones con unidades size:>40m<55m
status comprobar el texto de la etiqueta de estado status:failed
